## 恒毅校友会-保护眼睛

我们会以恒毅校友会的身份配合Ajo Optics于恒毅中学主办保护眼睛讲座会，提供免费验眼给b40家庭学生。
Ajo Optics将于当天提供基本验眼服务，并分发固本于学生，让学生可以各自到Ajo Optics分行进行更进一步的检查。
b40学生的固本将可以得到免费眼镜。
